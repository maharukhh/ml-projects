# Age Group Prediction System

A simple Python application that classifies a person's age into different age groups using predefined age ranges.

## Objective

The goal of this project is to determine the appropriate age category based on the user's input age.

## Age Categories

* Child (0–12 years)
* Teenager (13–19 years)
* Adult (20–59 years)
* Senior Citizen (60+ years)

## Technologies Used

* Python

## How It Works

1. User enters their age.
2. The program checks the corresponding age range.
3. The age group is identified.
4. The result is displayed to the user.

## Run the Project

```bash
python main.py
```
