# Employee Salary Prediction

A simple Machine Learning project that predicts employee salary based on years of experience using Linear Regression.

## Objective

The goal of this project is to demonstrate how Linear Regression can be used to predict salary growth based on work experience.

## Technologies Used

* Python
* NumPy
* Scikit-learn

## Dataset

This project uses a small sample dataset created directly within the Python code.

| Experience (Years) | Salary (Rs.) |
| ------------------ | ------------ |
| 1                  | 30000        |
| 2                  | 35000        |
| 3                  | 40000        |
| 4                  | 45000        |
| 5                  | 50000        |
| 6                  | 55000        |
| 7                  | 60000        |
| 8                  | 65000        |

## How It Works

1. Load experience and salary data.
2. Train a Linear Regression model.
3. Ask the user to enter years of experience.
4. Predict the expected salary.
5. Display the result.

## Run the Project

```bash
python main.py
```

## Example
