## Installation

Clone the repository:

```bash
git clone https://github.com/maharukhh/ml-projects.git
cd ml-projects
```

Install required packages:

```bash
pip install -r requirements.txt
```


## Running a Project

Navigate to any project folder and run the Python file:

```bash
cd project-folder
python main.py
```

Project-specific instructions can be found in each project's README file.
