# Simple Linear Regression

A Machine Learning project that demonstrates Simple Linear Regression using Scikit-learn to predict values and visualize the relationship between input and output data.


## Objective

The objective of this project is to understand how a Linear Regression model learns from data and predicts new values based on the learned relationship.


## Dataset

This project uses a small sample dataset created directly within the Python code.

| Input (X) | Output (Y) |
| --------- | ---------- |
| 1         | 2          |
| 2         | 4          |
| 3         | 6          |
| 4         | 8          |
| 5         | 10         |

## Technologies Used

* Python
* NumPy
* Scikit-learn
* Matplotlib

## Project Workflow

1. Create the sample dataset.
2. Train the Linear Regression model.
3. Predict the output for a new input value.
4. Display the model's slope and intercept.
5. Visualize the regression line using Matplotlib.

## Run the Project

```bash
python main.py
```
